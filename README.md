## 📖 项目简介
Dimensions（维度探索）是一款沉浸式、零门槛的高维空间科普单页应用。项目以现代前端技术为载体，将弦理论、M理论中晦涩抽象的0-11维时空概念，转化为可视化动画效果、通俗语言解读与交互式体验，让每一位对宇宙、物理、时空感兴趣的普通人，都能轻松开启一场跨越维度的探索之旅。

从零维的「奇点」到十一维的「M理论之巅」，项目通过精心设计的视觉动效、生活化的类比解读、体系化的知识梳理，打破专业物理与大众认知的壁垒，让高维时空不再是遥不可及的学术概念。

## 🚀 功能特性
### 1. 极致的沉浸式视觉体验
- 🌠 **高性能星空流星雨背景**：基于Canvas实现的全屏星空与随机流星动画，帧率优化，不影响页面滚动与交互流畅度
- 🔮 **维度专属可视化动画**：为0-11维每个维度定制专属CSS3/JS动画，包括零维奇点闪烁、三维立方体3D旋转、四维时间涟漪、高维轨道运动等，直观呈现维度的核心特征
- 🎞️ **现代化UI设计**：采用玻璃拟态（Glassmorphism）卡片设计，搭配科幻风格的字体与图标，层次分明，视觉冲击力强
- 🌓 **双视觉模式切换**：支持默认/增强视觉模式一键切换，提供不同的色彩与动效体验，适配不同的浏览偏好

### 2. 体系化的维度科普内容
- 📚 **全维度完整解读**：完整覆盖0维到11维的全维度体系，每个维度均包含核心定义、生活化形象比喻、现实实例、趣味冷知识四大模块，从零开始构建维度认知
- 🔬 **前沿理论深度解析**：内置可展开的深度分析板块，系统讲解弦理论、M理论、卡拉比-丘流形、全息原理等六大理论支柱，打通基础概念到前沿物理的知识链路
- 🏛️ **专业内容严谨性**：科普内容基于现代理论物理的主流认知编写，兼顾通俗性与专业性，避免错误科普
- 📖 **渐进式内容结构**：内容遵循从易到难的认知规律，从基础的零维点线面，逐步进阶到十一维M理论之巅，符合新手的学习路径

### 3. 丰富的交互式体验
- 🧪 **维度知识测验**：内置5道精选维度知识测试题，答题后即时反馈对错与解析，帮助用户检验学习成果，巩固知识点
- 🧭 **全维度快速导航**：桌面端常驻侧边导航栏、移动端适配弹窗式导航，支持一键直达任意维度板块，无需长页面滚动
- 📊 **实时阅读进度条**：页面顶部内置进度条，实时显示当前阅读进度，清晰掌握探索旅程的完成度
- ⚡ **丝滑的页面动效**：全页面滚动平滑过渡、元素入场动画、hover交互反馈，打造流畅丝滑的浏览体验

## 🏗️ 项目架构
项目采用**纯前端分层架构**，单文件内实现模块解耦，确保代码的可读性、可维护性与可扩展性，核心分层如下：
1. **数据层 (Data Layer)**
   - 维度核心数据：0-11维的所有科普内容、定义、比喻、实例数据
   - 理论知识库：弦理论、M理论等前沿物理的解析内容
   - 测验题库：知识测验的题目、选项、答案与解析数据
   - 配置项：全局动画、UI、交互的统一配置管理

2. **逻辑层 (Logic Layer)**
   - 页面交互逻辑：导航、模式切换、进度条、测验答题等核心交互控制
   - 动画控制逻辑：Canvas星空动画、维度专属动画的启停、帧率优化控制
   - 响应式适配逻辑：不同屏幕尺寸下的布局、交互模式切换逻辑
   - 性能优化逻辑：滚动节流、动画懒加载等性能优化逻辑

3. **视图层 (View Layer)**
   - 语义化页面结构：头部、导航、主内容区、维度板块、测验区、页脚的模块化结构
   - 响应式布局：基于Tailwind CSS实现的全端适配布局
   - UI组件：玻璃拟态卡片、折叠面板、按钮、导航等可复用UI组件

4. **动画层 (Animation Layer)**
   - Canvas动画：星空、流星等全屏背景动画
   - CSS3动画：维度可视化、元素过渡、hover动效等
   - JS驱动动画：滚动触发的入场动画、时间线动画等

## 📦 快速开始
项目无需任何环境依赖与后端服务，仅需两步即可在本地运行：

### 方式一：直接下载运行
1. 下载项目根目录下的 `index.html` 文件到本地
2. 直接用Chrome、Edge、Firefox等现代浏览器打开该文件，即可开启维度探索之旅

### 方式二：克隆仓库运行
1. 克隆本仓库到本地
```bash
git clone https://github.com/badhope/Dimensions.git
```
2. 进入项目目录
```bash
cd Dimensions
```
3. 用浏览器打开 `index.html` 文件，即可运行项目

### 部署说明
- 本地部署：可使用任何静态文件服务器（如Nginx、Apache、VS Code Live Server插件）托管项目
- 在线部署：支持一键部署到GitHub Pages、Vercel、Netlify、Cloudflare Pages等所有静态网站托管平台
